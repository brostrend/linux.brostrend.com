Linux support for BrosTrend WiFi Adapter

To request Linux support for our AC1 or AC3 models, please send an email to linux@trend-tech.net.cn, including the following information. Use Alt+Ctrl+T or the menus to open a terminal, then run all the following commands, even if you had run them again in the past:

# These execute the installer, so that we see the error messages:

wget deb.trendtechcn.com/installer.sh -O /tmp/installer.sh

sh /tmp/installer.sh


# Now (remove and) insert the adapter. If it doesnâ€™t work, continue.

# These verify that the adapter was inserted and the module loaded:

lsusb | grep 8812

lsmod | grep 8812 || sudo dpkg-reconfigure rtl8812au-dkms && sudo modprobe 8812au


# These show your kernel and distribution name:

uname -a

grep PRETTY /etc/os-release


# These check if the correct headers are installed:

ls /lib/modules

dpkg -l "*$(uname -r)*" | grep "$(uname -r)"

After these steps, please select all the text that is shown on the terminal with the mouse, including the commands and their output. Right click and Copy the text, then go to your mail application and paste the text inside the message body, without creating any attachments.

Common problems

  • Currently, our driver only supports kernels up to 4.18.x, which is what Ubuntu 18.04.2 ships with. Newer kernels like 4.19 or 5.x will be supported later on by a newer driver from Realtek.

  • If the driver was correctly loaded, but the signal is weak, or the access point keeps asking for a password, please also include the output of the following command:

    iwlist scanning
