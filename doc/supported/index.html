Supported distributions

The AC1/AC3 adapters are supported in the following distributions:

  • Ubuntu 16.04 (Xenial) and later

  • Debian 9 (Stretch) and later

  • Raspbian 9 (Stretch) and later

The adapters should also work in many more Debian and Ubuntu based distributions, like LinuxMint. A table follows with a list of distributions where the AC1/AC3 adapters were successfully tested.
